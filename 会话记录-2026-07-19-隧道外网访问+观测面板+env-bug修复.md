## 二、关键决策

- **Codfair = Cloudflare**（"解析"=DNS，"安全模式外部访问"=Tunnel/Zero Trust，唯一对得上）。

- **面板用纯 Python 标准库**（`http.server`），Spark 上零新依赖免 pip。
- **一条隧道 + 三子域名 + 一层反代**，最简。

## 八、★根治 bug：.env 行内注释污染环境变量致 ComfyUI InvalidURL

**现象**（用户飞书发「spark 状态」）：`🔴 ComfyUI — http://127.0.0.1:8188          # aeon-spark ComfyUI（生图引擎） 不可达（InvalidURL）`——URL 里混进了 `.env` 的行内注释。

**根因**：Spark 上 `.env` 由 `cp .env.spark.example .env` 而来，example 是 `COMFY_URL=http://...   # 注释` 行内注释格式。
`systemd/mgj-spark.service` 用 `EnvironmentFile=` 加载 .env，而 **systemd 不剥行内 `#` 注释**（bash 的 `source` 会剥，所以 start.sh 手跑没事、systemd 守护中招）→ COMFY_URL/SPARK_DEPLOY_DIR 整段含注释 → InvalidURL。
**为何面板绿、飞书红**：mgj-board 服务不加载 .env，走干净默认值；bridge(飞书) 经 systemd 加载脏 .env。

**修复（双保险 + 上机验证）**：
1. `.env.spark.example`：所有行内注释挪到独立行（`source` 与 systemd 都安全）。
2. `plugins/spark_ops.py` + `core/comfyui_client.py`：env 读取 `.split("#",1)[0].strip()` 剥注释+空白（脏 .env 也不崩）。
3. `observe/board.py`：tick() 用 `location.origin`（见五）。
4. **上机**：重写干净 .env + 重启 mgj-spark/mgj-board → `collect_all()` 实测 `True ComfyUI(aeon-spark) | cuda:0 NVIDIA GB10 · VRAM 122G` —— **ComfyUI 转 🟢**。
