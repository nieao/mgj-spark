# 部署说明

> 明确说明：项目如何利用本地算力部署智能体、如何优化大模型以适配 DGX Spark。

## 一、总体拓扑

本项目使用 **2 台 DGX Spark FE**（GB10 / ARM64 / 128GB 统一内存），职责分离：

- **A 机（大模型节点）**：部署 StepFun step-3.7-flash 本地量化版，提供纯本地 LLM API（意图定档 / 多步规划 / 兜底问答）。
- **B 机（生成 + 总控节点）**：部署 aeon-spark ComfyUI（Flux2 / Qwen-Image-Edit / LTX-2.3）、mgj-spark 安全总控、观测看板、Cloudflare 隧道。

两机避免资源争抢、降低延迟；理想态下可用 QSFP112 DAC 高速线缆双机互联跑更大模型。

## 二、智能体如何用本地算力部署

1. **送码上机**：代码经 scp / paramiko 送上 Spark，`python3 -m venv .venv && pip install -r requirements.txt`（纯 Python 依赖，ARM64 原生装，无需 Node）。
2. **填凭证**：`python setup.py` 生成凭证模板，填飞书 app 三件套 + StepFun key（真实密钥永不入库，均以 `*.example` 模板给出）。
3. **体检启动**：`bridge.py --selftest` 全绿即就绪 → `systemctl enable --now mgj-spark` 交给 systemd 守护，WebSocket 断线自愈、进程崩溃自动重启。
4. **三层守护全开机自启**：mgj-spark 总控 / AIGC-spark 生图工作台 + ComfyUI / Cloudflare 隧道 + Caddy，各自独立 systemd 单元。

**真机验证状态**（自测快照）：
- `mgj-spark`（飞书总控）、`mgj-board`（观测看板 :8250）— systemd `active (running)`；
- ComfyUI `:8188`、AIGC 工作台 `:8265`、观测看板 `:8250` — 健康端点 HTTP 200；
- ComfyUI 容器 `Up (healthy)`；GPU：NVIDIA GB10，空闲态；
- 飞书双向连通：入站消息正常接收、出站卡片可推达、非本人消息自动过滤。

## 三、大模型如何在本地优化 / 适配 Spark

- **StepFun step-3.7-flash 量化部署**：
  - 理想态（双机）：参考 MiaAI-Lab 的 `Dual-DGX-Spark-Step-3.7-Flash-NVFP4` 方案，以 **NVFP4** 精度双节点部署，利用 GB10 的 NVFP4 硬件加速，合理功耗下达平均 **25 token/s**；
  - 现实态（单机）：受硬件所限改为一台上部署 **Q3_K_M（94GB）** 低精度量化版本，吃满 128GB 统一内存容量优势，性能/精度略降但满足全本地运作，为 Agent 提供本地 API 接入与调度支持。
- **LTX-2.3 22B 视频模型**：采用 **fp8 权重 + distilled LoRA**，配合统一内存单机跑通 22B 视频全链路（checkpoint + Gemma-3-12B 编码器 + VAE + 空间/时间放大器同时常驻）。
- **ComfyUI 适配 GB10**：使用针对 GB10 优化的 aeon-spark（NVFP4）版本；容器化部署换取 ARM 依赖确定性；长下载/长任务可续传自愈（HF 后端卡死用 `docker restart` 断点续传恢复）。
- **工作流复用提效**：复用 `/history` 黄金工作流拍平成 API，逐段注入参数，避免盲拼节点反复烧 GPU。
- **对话侧本地/云端动态择优**：本地 StepFun 与赛事公网 API 之间，在低延迟、更高精度、按量、免占本地显存之间动态选择；资源受限时把 GB10 算力全部让给生图/视频扩散模型。

## 四、网络与安全部署

- **零端口映射外网**：`cloudflared` 只出站隧道 + Caddy 反代 Basic Auth 登录门，家宽 NAT 下无需在公网暴露服务器端口。
- **端口收敛**：关闭 Spark 上除必要端口外的所有公网端口，通信全走 Cloudflare Tunnel。
- **凭证加固**：修改配置中的默认 Token / Access Key 提高复杂度；`config/credentials.json`、`.env` 等敏感文件 `.gitignore` 强制不入库。
